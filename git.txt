cd (diretório) > vai para pasta selecionada
cd .. > volta uma pasta
ls > mostra lista de todas os diretórios
cntrl L > apaga o histórico



git status >mostra o status
git clone > faz o download de um repositório do github
git add > . ou -A adiciona (TODOS) os arquivos para o repositório local, em estado de stage
git commmit -m "comentário sobre a mudança no arquivo" > prepara o arquivo para ser mandado para a nuvem com um comentário
git push > envia o arquivo para o GitHub
